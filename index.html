CareLink – Full-Stack NYC Resource & Community Hub

CareLink is a production-ready full-stack platform designed to bridge the gap between New Yorkers and social services. Built with a decoupled MERN architecture, the platform aggregates real-time NYC Open Data and global content into a high-performance interface with persistent user data.

  • Database & Persistence: Integrated MongoDB Atlas to manage user profiles, authentication, and persistent storage for saved resources, utilizing Mongoose for schema validation and efficient querying.
  • System Architecture: Engineered a RESTful Express backend to serve as a secure proxy layer for NewsAPI and YouTube v3 APIs, implementing server-side data normalization and credential protection.
  • UI/UX Engineering: Developed a modern Glassmorphism-styled interface in React, featuring Context API for global state management and protected authentication routes.
  • Cloud Deployment: Successfully deployed a distributed system with the Frontend on Vercel and Backend on Render, managing production-grade environment variables and cross-origin resource sharing (CORS).
